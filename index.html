Admin / Staff Panel — Coming Soon

Let Coffee Life staff update this menu in real time.

This section will become the internal panel for Coffee Life in Boise. From here, owners and staff will be able to manage the drinks without touching code.

  • Add new drinks with name, size, price, and category.
  • Retrieve and display all entries from the AWS database.
  • Delete drinks that are sold out or out of season.
Future functionality:
• “Add New Drink” form that sends data to AWS
• “View All Drinks” button that loads entries into these cards
• “Delete” action per drink

For now, this is a visual placeholder so the landing page already communicates how the admin tools will work once the backend is wired up.

Add a New Drink (AWS Placeholder)

This form is wired to placeholder JavaScript functions for now. In the next step, it will submit to an AWS API (API Gateway + Lambda + DB).

View All Drinks (AWS Placeholder)

This will eventually call an AWS endpoint to pull every drink from the database. For now it just logs to the console and shows a message.

When wired to AWS, drinks returned from the API will be rendered here.
